# Изображения

## SVG карточки услуг

Для корректной работы секции "Услуги" необходимо скопировать следующие файлы из родительской папки в эту директорию:

1. `карточка услуг 1.svg` → `service-card-1.svg`
2. `карточка услуг 2.svg` → `service-card-2.svg`
3. `карточка услуг 3.svg` → `service-card-3.svg`

### Инструкция по копированию:

**Windows PowerShell:**
```powershell
Copy-Item "..\..\карточка услуг 1.svg" "service-card-1.svg"
Copy-Item "..\..\карточка услуг 2.svg" "service-card-2.svg"
Copy-Item "..\..\карточка услуг 3.svg" "service-card-3.svg"
```

**Или вручную:**
Скопируйте файлы из папки `c:\Users\Администратор\Desktop\верстка лендинга\` в папку `landing-barber\public\images\` с переименованием как указано выше.
